# 短视频拉片分析助手

一个帮你拆解爆款短视频、提炼创作方法论的 SOLO Skill。

## 这是什么

做短视频的朋友应该都有这种经历：刷到一个爆款，很想知道它为什么能火，但手动拉片太费时间——要逐帧截图、记时间轴、分析节奏...一个视频搞下来两三个小时没了。

这个 Skill 就是来解决这个问题的。你把视频丢给它，它能自动从叙事结构、情绪曲线、镜头语言、剪辑节奏等维度拆解，最后输出一份可以直接用的分析报告。

## 能做什么

- **自动识别视频类型**：剧情短剧、知识科普、带货种草、情绪短片...
- **多维度拆解**：不只是告诉你"讲了什么"，而是分析"怎么讲的"
- **输出可复用的方法论**：不是看完就忘，而是能用到自己的创作里
- **5种输出模式**：标准报告、爆款机制、分镜表、创作模板、优化建议

## 使用方法

### 直接丢链接
```
帮我拉片分析一下这个视频：https://www.bilibili.com/video/xxx
```

### 指定输出格式
```
分析一下这个视频为什么能爆
```

```
给我输出分镜表，我要学拍摄
```

```
提炼成可复用的创作模板
```

### 提供素材
如果视频链接打不开，可以提供：
- 字幕/文案文本
- 关键画面截图
- 弹幕数据
- 视频描述

## 怎么装

```bash
npx skills add /path/to/short-video-shot-analysis -g
```

或者直接把 `SKILL.md` 放到你的 SOLO skills 目录。
