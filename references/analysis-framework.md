# 短视频/短剧分析框架

## 1. 内容类型判断

### 1.1 短剧 (Short Drama)
- 特征：连续剧情、角色关系、冲突推进、悬念卡点
- 常见题材：霸总、逆袭、复仇、甜宠、家庭伦理、悬疑
- 分析重点：人物关系、 stakes（赌注/代价）、爽点、反转、结尾钩子

### 1.2 剧情短视频 (Plot Short Video)
- 特征：完整故事、3-5分钟、有起承转合
- 常见形式：一人分饰多角、情景剧、反转段子
- 分析重点：故事结构、节奏、笑点/泪点、信息差

### 1.3 产品视频 (Product Video)
- 特征：展示产品、解决问题、促转化
- 常见形式：测评、对比、使用场景、效果展示
- 分析重点：痛点-解决方案、信任背书、欲望刺激、转化触发

### 1.4 知识视频 (Knowledge Video)
- 特征：信息传递、教育、科普
- 常见形式：口播、动画、图文混排
- 分析重点：好奇心缺口、信息密度、结构清晰度、记忆点

### 1.5 口播视频 (Talking Head)
- 特征：真人出镜、直接对话、观点输出
- 常见形式：经验分享、观点表达、情感共鸣
- 分析重点：人设、表达节奏、情绪感染、观点锐度

### 1.6 品牌广告 (Brand Ad)
- 特征：品牌调性、情感连接、记忆点
- 常见形式：故事片、概念片、用户证言
- 分析重点：品牌信息、情感共鸣、传播性

### 1.7 Vlog/Lifestyle
- 特征：真实记录、生活方式、个人IP
- 常见形式：日常记录、探店、旅行、学习工作
- 分析重点：真实感、向往感、节奏、人设一致性

## 2. 叙事结构分析

### 2.1 经典三幕结构
```
第一幕（铺垫）：背景 + 冲突引入（0-20%）
第二幕（对抗）：冲突升级 + 阻碍（20-80%）
第三幕（解决）：高潮 + 结局（80-100%）
```

### 2.2 短视频专用结构
```
钩子（0-3秒）→ 铺垫（3-10秒）→ 冲突（10-30秒）→ 升级（30-50秒）→ 反转/爽点（50-55秒）→ 结尾/引导（55-60秒）
```

### 2.3 短剧单集结构
```
上集回顾/悬念承接 → 冲突升级 → 新信息/反转 → 情绪顶点 → 下集预告/卡点
```

### 2.4 分析要点
- **起点**：观众从什么状态进入故事
- **触发事件**：什么打破了平衡
- **上升动作**：冲突如何逐步升级
- **高潮**：最大张力点在哪里
- **结局**：如何释放张力
- **结尾钩子**：如何引导继续观看

## 3. 情绪曲线分析

### 3.1 情绪维度
- **基础情绪**：喜、怒、哀、惧、惊、厌
- **复合情绪**：爽（期待满足）、尬（社交不适）、甜（浪漫满足）、燃（热血激动）

### 3.2 情绪曲线类型
| 类型 | 特征 | 适用内容 |
|---|---|---|
| 上升型 | 情绪逐步累积，最后释放 | 励志、逆袭、复仇 |
| 波动型 | 情绪起伏，有多个小高潮 | 剧情、悬疑 |
| 陡降陡升 | 先抑后扬，强对比 | 反转、爽剧 |
| 平稳型 | 情绪稳定，靠信息推进 | 知识、科普 |

### 3.3 分析要点
- **起点情绪**：观众最初的情绪状态
- **情绪触发点**：什么引发了情绪变化
- **情绪强度**：变化有多强烈
- **情绪节奏**：变化频率和幅度
- **情绪释放**：如何满足或释放情绪

## 4. 视觉层分析

### 4.1 景别
| 景别 | 范围 | 功能 |
|---|---|---|
| 远景 | 人物全身+环境 | 交代环境、氛围 |
| 全景 | 人物全身 | 展示动作、关系 |
| 中景 | 膝盖以上 | 对话、互动 |
| 近景 | 胸部以上 | 表情、情绪 |
| 特写 | 面部/局部 | 强调细节、情绪放大 |
| 大特写 | 眼睛/物体局部 | 极致强调、悬念 |

### 4.2 镜头运动
- **推**：强调重点、情绪放大
- **拉**：揭示环境、关系变化
- **摇**：展示空间、跟随动作
- **移**：跟随主体、营造动感
- **跟**：保持主体在画面中的位置
- **升降**：改变视角、营造氛围

### 4.3 构图
- **三分法**：主体放在交叉点
- **中心构图**：强调主体、正式感
- **对称构图**：平衡、秩序、美感
- **框架构图**：引导视线、增加层次
- **引导线**：引导观众视线

### 4.4 光影与色彩
- **光质**：硬光（戏剧感）vs 柔光（自然感）
- **光位**：顺光、侧光、逆光、顶光
- **色调**：暖调（温馨/紧张）、冷调（冷静/疏离）
- **色彩对比**：互补色、冷暖对比

## 5. 剪辑层分析

### 5.1 剪辑节奏
- **快剪**：0.5-2秒/镜，紧张、刺激、信息量大
- **中速剪**：3-5秒/镜，正常叙事
- **慢剪**：5秒以上/镜，强调、情绪、氛围

### 5.2 转场类型
| 转场 | 效果 | 适用场景 |
|---|---|---|
| 硬切 | 直接、节奏快 | 大多数短视频 |
| 淡入淡出 | 柔和、时间流逝 | 情绪过渡 |
| 叠化 | 回忆、时间跳跃 | 闪回、梦境 |
| 划像 | 活泼、趣味 | 轻松内容 |
| 匹配剪辑 | 流畅、专业 | 动作连贯 |
| J-cut/L-cut | 声音先入/后出 | 对话、叙事流畅 |

### 5.3 剪辑技巧
- **跳切**：压缩时间、增加节奏
- **交叉剪辑**：并行叙事、增加张力
- **匹配剪辑**：视觉连贯、专业感
- **节奏剪辑**：根据音乐节拍剪辑

## 6. 字幕/文案层分析

### 6.1 字幕类型
- **对白字幕**：同步对话
- **强调字幕**：放大关键词
- **解说字幕**：补充信息
- **情绪字幕**：表达内心OS
- **引导字幕**：引导互动

### 6.2 字幕设计
- **字体**：风格匹配内容调性
- **颜色**：对比度、品牌色、情绪色
- **动效**：出现方式、强调效果
- **位置**：不遮挡主体、符合阅读习惯

### 6.3 文案技巧
- **数字**：具体、可信
- **痛点**：引发共鸣
- **悬念**：引发好奇
- **对比**：突出差异
- **金句**：易传播、易记忆

---

## 7. 转化层分析（产品/营销视频）

### 7.1 转化漏斗
```
注意 → 兴趣 → 欲望 → 信任 → 行动
```

### 7.2 关键要素
- **痛点共鸣**：让用户觉得"说的就是我"
- **解决方案**：清晰展示如何解决问题
- **信任证明**：数据、案例、权威背书
- **稀缺性**：限时、限量、独家
- **行动指令**：明确告诉用户做什么

---

## 8. 留存层分析

### 8.1 完播设计
- **信息缺口**：设置悬念，让观众想知道答案
- **节奏控制**：避免拖沓，保持信息密度
- **情绪承诺**：暗示后面有更大的情绪释放
- **视觉变化**：定期改变画面，防止疲劳

### 8.2 互动设计
- **评论引导**：提出争议性问题
- **点赞引导**：情感共鸣点
- **分享引导**：社交货币、身份表达
- **关注引导**：系列内容预告

### 8.3 平台适配
- **抖音**：前3秒决定生死，节奏快，信息密度高
- **快手**：真实感、人设、互动性
- **B站**：深度、梗、弹幕文化
- **视频号**：社交传播、中老年用户适配
